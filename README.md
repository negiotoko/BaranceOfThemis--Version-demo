​テミスの天秤：ルールブック
​1. ゲームの勝利条件
​総合勝利
先に指定された回数のラウンドを取得したプレイヤーが、総合勝利となります。  
​ラウンド勝利
各ラウンドの勝利条件は、相手が隠した3枚の「トゥルーカード」が示す3桁の数字を、先に正確に当てることです。  
​2. ゲームの準備
​ゲーム開始時、お互いに2種類のデッキを持ちます。  
​トゥルーデッキ: 「1」～「9」の数字カード。  
​バトルデッキ: 数字カードや様々な特殊カード。  
​ラウンドが始まると、以下の準備を行います。
​トゥルーカード選択: お互いに、トゥルーデッキから数字カード3枚を「トゥルーカード」（そのラウンドの「答え」）として、A、B、Cレーンに1枚づつ伏せます。  
​バトルカード選択: お互いに、バトルデッキからカード3枚を「バトルカード」として、A、B、Cレーンに１枚づつ伏せます。  
​3. ゲームの流れ：ジャッジ（バトル）
​準備が終わると、あなたのターンからゲームが開始されます。  
​ジャッジの宣言
自分のターンに、3つのレーンのうち1つを選んで「ジャッジ」を宣言します。
​勝敗の判定
ジャッジしたレーンのお互いのバトルカードを表にし、値を比べます。
​数字カード: 書かれている数字がそのまま値になります。
特殊カード:　特殊カードの効果によって値が決まります。
​値が高い方がジャッジの勝者となります。
​同点の場合:
値がまったく同じだった場合、ジャッジを仕掛けた側（現在のターンプレイヤー）の敗北となります。  
​4. ゲームの流れ：アクション
​ジャッジの勝者は「アクション権」を獲得し、「解答する」か「質問する」かを選びます。  
​A. 解答する
相手のトゥルーカードが示す3桁の数字を宣言します。  
​正解: あなたのラウンド勝利です。  
​不正解: ターンが相手に移り、補充フェーズへ移行します。  
​B. 質問する
ジャッジに勝利したレーンと同じレーンの、相手のトゥルーカード（答えの1枚）について、以下の5種類から質問できます。  
​奇数か？
​偶数か？
​〇以上か？
​〇以下か？
​〇と等しいか？
相手から答えを聞いたら、ターンが相手に移り、補充フェーズへ移行します。  
​5. ゲームの流れ：補充とターン交代
​「質問」を選んだり、「解答」で不正解だったりした場合、ターンが移る前に以下の処理が行われます。
​補充
ジャッジで使われたレーンのバトルカードが両方とも破棄されます。その後、お互いに手札から新しいカードを1枚選び、空いたレーンに補充します。  
​ターン交代
補充が完了すると、相手のターンに移行します。 

​6. 手札が尽きた場合
​お互いの手札が0枚になると、ルールが少し変わります。  
​補充の停止:
この状態では、ジャッジで使われたカードは補充されず、破棄されるだけになります。  
​カードのリセット:
お互いの場にあるバトルカードが１枚のみになる（一つのレーン以外にバトルカードが存在しない状態になる）と、全カードがリセットされ、手札が配り直されて「2. ゲームの準備」のバトルカード選択に戻ります。  
​禁止ルール:
手札が尽きた後、カードが置かれていない（空の）レーンをジャッジすることはできません。
